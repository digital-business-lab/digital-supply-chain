# Coffee House Island — Consumer Endpoint

The lean end of the supply chain. No ERP, no Kafka, no Fabric peer. Three independent modules — POS, Traceability Display, IoT connector — can run on one machine or separately. IoT processing is delegated to the Lab Cloud.

📖 **[Full documentation → GitHub Pages](https://digital-business-lab.github.io/digital-supply-chain/islands/coffeehouse/)**

---

## Hardware at a Glance

| Component | Function |
|---|---|
| Linux PC | POS module + Traceability Display |
| RFID / barcode scanner | Goods receipt and customer receipts |
| Customer-facing display | Traceability Display |
| Smart coffee machine | IoT sensors → Lab Cloud |

## Modules

POS System · Traceability Display · IoT Connector (→ Lab Cloud)
